# comp-meeting-js

Competence meeting for JS and TS

### Install

`yarn install`
or
`npm install`

### Test

`yarn test`
or
`npm test`

## Topics

1. Functions
2. Let
3. Default Values
4. Const
5. Spread/deconstruct
6. Templates
7. Arrays - reduce, map, filter, sort
8. Promises, Async/Await
9. Classes
